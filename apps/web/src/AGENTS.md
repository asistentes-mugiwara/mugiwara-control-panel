# AGENTS.md — apps/web/src

## Rol
Código fuente del frontend.

## Reglas
- Organizar por app/modules/shared con límites claros.
- Evitar mezclar componentes de dominio con utilidades globales.
- **Vigilar `.gitignore`** y no introducir artefactos generados.
- Documentar cambios estructurales en este `AGENTS.md` y en docs.
- Cuando se escriba código de frontend, consultar y respetar `docs/frontend-ui-spec.md` como contrato de referencia de shell, pantallas, estados y componentes compartidos.
- Para naming, tokens, assets y orden de implementación, consultar también `docs/frontend-implementation-handoff.md`.
