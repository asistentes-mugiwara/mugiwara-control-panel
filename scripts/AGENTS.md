# AGENTS.md — scripts

## Rol
Automatizaciones seguras del repositorio.

## Reglas
- Scripts explícitos, revisables y no mágicos.
- No introducir automatizaciones que ejecuten acciones destructivas sin protección.
- **Vigilar `.gitignore`** para no subir outputs temporales o credenciales locales.
- Documentar inputs, outputs y riesgos de cada script.
